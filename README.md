# onuw-helper
一夜終極狼人殺語音小幫手 (ONUW Voice Helper)

## 本地開發啟用方式

### 1. 環境需求
- Node.js 18 以上（建議使用 LTS）
- npm 9 以上

### 2. 安裝依賴
```bash
npm install
```

### 3. 啟動開發伺服器
```bash
npm run dev
```

啟動後開啟終端機顯示的本地網址（通常是 `http://localhost:5173`）。

### 4. 常用指令
```bash
# 型別檢查 + 正式版打包
npm run build

# 本地預覽打包結果
npm run preview
```
